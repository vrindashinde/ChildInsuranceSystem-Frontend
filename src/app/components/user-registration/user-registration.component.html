User Registration

First Name is required and should consist of 2 to 50 characters (letters only).

Last Name is required and should consist of 2 to 50 characters (letters only).

Aadhar Number is required and should consist of 12 digits.

Address is required.

Phone Number is required and should match the pattern (10 digits).

Email is required and should be a valid email address.

Password is required and should match the pattern (6-12 characters, at least one lowercase letter, one uppercase letter, and one digit).

Bank Name is required and should be between 2 to 50 characters (letters, spaces allowed).

Account Number is required and should be between 9 to 18 digits.

IFSC Code is required and should be in the format: ABCD0123456.
